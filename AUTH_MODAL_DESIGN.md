## Color Specifications

### Modal Background
- Background Color: `#0b0b0b` (very dark gray/black)
- Border Radius: `rounded-2xl` (16px)
- Shadow: Large shadow for depth

### Text Colors
- **Primary Text** (Headings): `#ffffff` (white)
- **Secondary Text** (Descriptions): `rgba(255, 255, 255, 0.6)` (60% white)
- **Tertiary Text** (Placeholders): `rgba(255, 255, 255, 0.4)` (40% white)

### Input Fields
- Background: `rgba(255, 255, 255, 0.05)` (very faint white)
- Border: `rgba(255, 255, 255, 0.1)` (subtle border)
- Border on Focus: `#F37120` with 50% opacity
- Ring on Focus: `#F37120` with 20% opacity

### Buttons
- **Primary Button** (Sign In / Create Account)
  - Background: `#F37120` (bright orange)
  - Text: `#000000` (black)
  - Hover: `#E86010` (darker orange)
  - Disabled: `#F37120` with 60% opacity
  - Font Weight: semibold

- **Tab Button**
  - Inactive: `rgba(255, 255, 255, 0.6)` text
  - Active: `#F37120` background + black text
  - Border Radius: `rounded-md`

### Close Button
- Color: `rgba(255, 255, 255, 0.6)`
- Hover: `#ffffff`

### Overlay Background
- Color: `rgba(0, 0, 0, 0.6)` (60% transparent black)

---

## Typography

### Headings
- Font: Geist Sans (default)
- "Welcome Back" / "Join BearFit"
  - Size: 24px (text-2xl)
  - Weight: Bold
  - Color: White

### Descriptions
- Font: Geist Sans (default)
- Size: 14px
- Weight: Normal
- Color: `rgba(255, 255, 255, 0.6)`
- Example: "Sign in to access your dashboard"

### Labels
- Font: Geist Sans (default)
- Size: 14px
- Weight: Medium (600)
- Color: `rgba(255, 255, 255, 0.8)`
- Example: "Email"

### Input Placeholder Text
- Font: Geist Sans (default)
- Size: 14px
- Color: `rgba(255, 255, 255, 0.4)`
- Example: "you@example.com"

### Button Text
- Font: Geist Sans (default)
- Size: 14px
- Weight: Semibold (600)
- Letter Spacing: Normal

---

## Responsive Design

### Desktop (1024px+)
- Modal Max Width: 448px (28rem)
- Modal Centered on Screen
- Margin: 16px on each side

### Tablet (768px+)
- Modal Max Width: 448px
- Modal Centered with margins

### Mobile (< 768px)
- Modal Max Width: 100% - 32px
- Modal Centered with 16px margin on each side
- Padding: 32px (p-8)

---

## Animations & Transitions

### Modal Appearance
- Overlay fades in
- Modal slides in from top

### Button Interactions
- Hover: Color transition (smooth)
- Disabled: Opacity reduces to 60%
- Loading: "Processing..." text

### Input Focus
- Border color transitions to orange
- Ring appears with orange glow
- Smooth transition

### Tab Switch
- Background color transitions smoothly
- No page reload or full re-render

---

## Accessibility Features

- Close button labeled: "Close modal"
- High contrast text: White on dark background
- Focus ring visible on all interactive elements
- Form labels associated with inputs
- Error messages in red for visibility
- Success messages in green for visibility
- Tab order follows form flow

## Browser Compatibility

- ✅ Chrome/Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Mobile browsers (iOS Safari, Chrome Mobile)

---

## Performance Notes

- Modal uses CSS transforms for smooth animations
- No unnecessary re-renders
- Form state managed efficiently
- API calls debounced
- Icons from Lucide (lightweight SVG)

---

## Dark Mode Compatibility

- ✅ Fully compatible with dark mode
- Uses explicit dark colors (not system theme)
- No light mode needed (intentional design choice)

---

## Known Limitations

- Modal doesn't persist form data if closed
- No remember me functionality (for security)
- Single browser tab session (not synced across tabs)
- Phone field optional, not validated server-side
